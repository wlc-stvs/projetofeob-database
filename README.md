Projeto de banco de dados para análise de vendas com foco em Business Intelligence.
Estruturado em MySQL, organiza dados de clientes, produtos e pedidos.
Permite consultas analíticas para identificar padrões de compra.
Pode ser integrado ao Power BI para visualização de indicadores.
Focado em transformar dados em suporte para tomada de decisão.
